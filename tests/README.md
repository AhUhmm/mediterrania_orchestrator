## Test Run

```bash
# Run all tests
pytest

# Run specific test file
pytest tests/test_nutritional_verifier.py

# Run specific test class
pytest tests/test_nutritional_verifier.py::TestVerificatoreNutrizionale

# Run specific test method
pytest tests/test_nutritional_verifier.py::TestVerificatoreNutrizionale::test_calcolo_requisiti_uomo
```
